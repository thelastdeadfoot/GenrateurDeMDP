Titre : stockage de mot de passe

En tant qu'utilisateur, 
Je souhaite que mes mots de passe soient stockés dans un environnement sécurisé (base de données), et pouvoir consulter cet environnement via l'application 

Critère d'acceptation
L'utilisateur doit pouvoir voir la base de données via l'application
L'utilisateur doit pouvoir exporter les mdp de cette base de données dans un fichier chiffré
Il doit pouvoir rechercher un MdP rapidement via cette base de données
L'utilisateur doit pouvoir importer des mots de passe sécurisés directement dans la BDD, un processus d'analyse se mettra en place avant l'insertion des mots de passe, si le mot de passe n'est pas assez sécurisé, un message d'erreur apparaîtra

Priorité : Haute

Note technique : 
utilisation d'une base de données en ligne
